# Tailwind

- Tạo ra các class tương ứng với thuộc tính và giá trị
- Chia rất nhỏ

## Cách học

- Xác định được thuộc tính CSS
- Tìm tiền tố trong tailwind
- Áp dụng vào cú pháp

## Cú pháp

1. Cơ bản

```
tiento-giatri (giatri có sẵn của tailwidn)
tiento-[custom-value] (custom-value là giá trị tự đưa vào)
```

2. Cú pháp Pseudo

```
tenpseudo:tiento-giatri
tenpseudo:tiento-[custom-value]
```

3. Cú pháp responsive

- Tuân thủ nguyên tắc mobile first
- Cú pháp

```
ten-breakpoint:tiento-giatri
ten-breakpoint:tiento-[custom-value]
```
